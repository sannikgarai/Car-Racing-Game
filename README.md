# 🚗 Car Racing Game (Python - Pygame)

## 📌 Description
A simple and interactive **2D Car Racing Game** built using **Python and Pygame**. The player controls a car to avoid incoming enemy vehicles while scoring points over time. The game becomes progressively challenging as speed increases, testing reflexes and coordination.

---

## 🚀 Features
- 🎮 Smooth car controls using keyboard (arrow keys)
- 🚗 Endless racing gameplay
- 💥 Collision detection (Game Over on crash)
- 📈 Score tracking system
- ⚡ Increasing difficulty over time
- 🎨 Simple and clean graphics

---

## 🛠️ Tech Stack
- Python
- Pygame Library
